# üöÄ Mandarin Sprint

Your 30-day accelerated path to learning conversational Mandarin Chinese.

Mandarin Sprint is a cross-platform mobile application built with Flutter, designed to help beginners learn the fundamentals of Mandarin Chinese in just 30 days. The app follows a structured, day-by-day curriculum that makes learning manageable and effective.

## ‚ú® Features

*   **Structured 30-Day Curriculum:** Progress from basic pinyin to conversational dialogues over 30 carefully planned lessons.
*   **Daily Lessons:** Each day introduces new vocabulary and a practical dialogue.
*   **Interactive Flashcards:** Master new words with a built-in flashcard system.
*   **Pronunciation Guide:** Listen to and practice tricky initials, sibilants, and vowels with audio playback.
*   **Tone Guide:** Understand and master the four main tones of Mandarin with clear examples.
*   **Customizable Themes:** Personalize your learning experience with multiple color themes.
*   **Cross-Platform:** Learn on the go with support for Android, iOS, and the Web. The web version is a fully functional Progressive Web App (PWA).

## üíª Getting Started

To run the app on your local machine, follow these steps:

1.  Ensure you have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
2.  Clone this repository.
3.  Navigate to the `flutter_app` directory:
    ```bash
    cd flutter_app
    ```
4.  Install the dependencies:
    ```bash
    flutter pub get
    ```
5.  Run the app on your desired platform (e.g., mobile emulator or web):
    ```bash
    flutter run
    ```
    To run specifically on the web:
    ```bash
    flutter run -d chrome
    ```
